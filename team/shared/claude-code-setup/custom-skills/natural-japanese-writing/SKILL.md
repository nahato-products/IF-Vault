# Natural Japanese Writing

AI が生成しがちな不自然な日本語を避け、読みやすい自然な文章を書くスキル。

## よくある AI 日本語の問題

| アンチパターン | 改善例 |
|--------------|--------|
| 「〜することが重要です」の多用 | 「〜が大切」「〜がポイント」 |
| 「まず最初に」 | 「まず」 |
| 「〜という観点から」 | 「〜の面では」 |
| 「非常に重要な点として」 | 「大事なのは」 |
| 「それでは、〜について説明します」 | いきなり本題 |
| 体言止めを全く使わない | 適度に体言止めを活用 |

## 読みやすい文章の原則

1. **1文1情報**: 1センテンスに1つの情報
2. **能動態優先**: 「〜された」より「〜した」
3. **接続詞を減らす**: 文頭の「また」「そして」を削る
4. **具体例**: 抽象論は例で補強
5. **読者視点**: 「なぜ読者が知りたいか」から書く
