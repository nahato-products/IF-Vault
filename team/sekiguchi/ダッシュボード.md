## ノート数推移

```dataview
TABLE length(rows) AS ファイル数
FROM "team/guchi"
WHERE file.name != "ダッシュボード"
GROUP BY file.folder
SORT length(rows) DESC
```

## タグ別集計

```dataview
TABLE length(rows.file.name) AS ファイル数
FROM "team/guchi"
WHERE tags
FLATTEN tags AS tag
GROUP BY tag
SORT length(rows.file.name) DESC
LIMIT 15
```

## ステータス別

FROM "team/guchi"
