# クイックスタート

15分で Obsidian Vault を使い始める手順。細かい説明は後回しにして、まず動かすことを優先する。

---

## 1. リポジトリをクローン（5分）

```bash
cd ~/Documents
git clone git@github.com:nahato-Inc/IF-Vault.git "Obsidian Vault"
```

> [!NOTE]
> SSH鍵の設定がまだの場合は [[02_セットアップガイド]] を参照。

## 2. Obsidian で開く（3分）

1. [Obsidian](https://obsidian.md/download) をインストール（まだの場合）
2. Obsidian を起動
3. 「Open folder as vault」を選択
4. `~/Documents/Obsidian Vault` を選ぶ
5. コミュニティプラグインを有効化（設定 → コミュニティプラグイン → 制限モードをオフ）

## 3. 自分のフォルダを作って作業開始（7分）

`team/` 配下に自分の名前でフォルダを作る。

```
team/あなたの名前/
├── README.md           ← ダッシュボード
├── daily/              ← デイリーノート
├── projects/           ← プロジェクト管理
├── notes/              ← 技術メモ
└── code-snippets/      ← コードスニペット
```

`team/guchi/README.md` をコピーしてテンプレートにするのが早い。

---

## 毎日の使い方

**朝**: `Cmd + P` → "Open today's daily note" でデイリーノートを開いてタスクを書く

**作業中**: メモを取る。10分ごとに自動で GitHub に保存される

**夕方**: デイリーノートに作業記録を追記して、明日の予定を書く

---

## 覚えるショートカット（3つだけ）

| ショートカット | やること |
|--------------|---------|
| `Cmd + P` | コマンドパレット（最重要） |
| `Cmd + O` | ファイル検索 |
| `Cmd + N` | 新規ノート作成 |
