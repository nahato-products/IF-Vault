# ⚡ クイックスタートガイド

**所要時間**: 15分で使い始められます！

---

## 🎯 3ステップで開始

### Step 1: リポジトリをクローン（5分）

```bash
# ターミナルで実行
cd ~/Documents
git clone git@github.com:nahato-Inc/IF-memo.git "Obsidian Vault"
```

### Step 2: Obsidianで開く（3分）

```
1. Obsidianを起動
2. "Open folder as vault"
3. ~/Documents/Obsidian Vault を選択
4. コミュニティプラグインを有効化
```

### Step 3: 作業開始（7分）

```
1. team/配下に自分のフォルダ作成
2. Cmd + P → "Open today's daily note"
3. 今日のタスクを書く
```

**✅ 完了！**

---

## 📝 毎日の使い方

### 朝
```
Cmd + P → "Open today's daily note"
```

### 作業中
```
メモを取る → 自動で10分ごとにGitHub保存
```

### 夕方
```
作業記録を追記 → 明日の予定を書く
```

## ⌨️ 覚えるショートカット（3つだけ）

1. **Cmd + P** - コマンドパレット
2. **Cmd + O** - ファイル検索
3. **Cmd + N** - 新規ノート

**これだけ覚えればOK！**
