## 📝 Obsidianとは？

**Obsidian = ローカル保存型のメモアプリ**

### 特徴
- 📁 **ファイルは全てMarkdown形式** → テキストエディタでも開ける
- 💻 **ローカル保存** → ネット不要、高速、プライバシー安心
- 🔗 **ノート同士をリンク** → 情報が有機的につながる
- 🎨 **プラグインで拡張** → コード実行、カレンダー、タスク管理など

### 他のツールとの違い

| ツール | データの場所 | オフライン | プラグイン |
|--------|-------------|-----------|-----------|
| Notion | クラウド | ❌ | ❌ |
| Google Docs | クラウド | ❌ | ❌ |
| **Obsidian** | **ローカル** | **✅** | **✅** |

## 🤔 なぜGitで管理するのか？

### メリット

#### 1️⃣ **自動バックアップ**
```
10分ごとに自動でGitHubに保存
→ PCが壊れても大丈夫
→ 誤削除しても復元可能
```

#### 2️⃣ **チーム共有**
```
あなたのメモ → GitHub → チームメンバーが閲覧
チームの知識が蓄積される
```

#### 3️⃣ **変更履歴の追跡**
```
いつ・誰が・何を変更したか全て記録
過去のバージョンに戻せる
```

#### 4️⃣ **複数デバイスで同期**
```
会社のPC ←→ GitHub ←→ 自宅のPC
どこからでもアクセス可能
```

### 実際の運用イメージ

```
あなた（PC）
  ↓ 10分ごとに自動保存
GitHub（クラウド）
  ↓ 同期
チームメンバー（閲覧・編集可能）
```

## 🚀 セットアップ手順

### 前提条件チェック

- [ ] Macを使用している
- [ ] GitHubアカウントがある（なければ[作成](https://github.com/join)）
- [ ] Gitがインストール済み（ターミナルで `git --version` を実行）

---

### Step 1: Obsidianのインストール（5分）

#### 1-1. Obsidianをダウンロード
```
https://obsidian.md/download
→ Mac版をダウンロード
→ アプリケーションフォルダに移動
```

#### 1-2. Obsidianを起動
```
初回起動時:
- "Create new vault" を選択
- 名前: 任意（例: "MyNotes"）
- 場所: Documents（推奨）
```

---

### Step 2: リポジトリのクローン（10分）

#### 2-1. ターミナルを開く
```bash
# アプリケーション → ユーティリティ → ターミナル
```

#### 2-2. SSH鍵の設定（初回のみ）

**既にSSH鍵がある場合はスキップ**

```bash
# 鍵の確認
ls ~/.ssh/id_ed25519.pub

# なければ作成
ssh-keygen -t ed25519 -C "your-email@example.com"
# Enter連打でOK

# 公開鍵をコピー
pbcopy < ~/.ssh/id_ed25519.pub
```

#### 2-3. GitHubにSSH鍵を登録

```
1. https://github.com/settings/keys にアクセス
2. "New SSH key" をクリック
3. Title: "会社のMac"
4. Key: ペースト（Cmd + V）
5. "Add SSH key" をクリック
```

#### 2-4. リポジトリをクローン

```bash
# Documentsフォルダに移動
cd ~/Documents

# 既存のリポジトリをクローン
git clone git@github.com:nahato-Inc/IF-memo.git "Obsidian Vault"

# 完了確認
cd "Obsidian Vault"
ls
```

---

### Step 3: Obsidianでリポジトリを開く（3分）

#### 3-1. Obsidianで既存のVaultを開く
```
1. Obsidianを起動
2. 左下の "Open another vault" をクリック
3. "Open folder as vault" を選択
4. ~/Documents/Obsidian Vault を選択
```

#### 3-2. コミュニティプラグインを有効化
```
1. 設定（⚙️）を開く
2. コミュニティプラグイン → 制限モードをオフ
3. 確認ダイアログで "オフにする" をクリック
```

**✅ これで全てのプラグインが有効化されます！**

---

### Step 4: 動作確認（5分）

#### 4-1. Git同期の確認

```
1. Cmd + P でコマンドパレットを開く
2. 「git」と入力
3. "Obsidian Git: Create backup" が表示されればOK
```

#### 4-2. テストノートの作成

```
1. Cmd + N で新規ノート作成
2. タイトル: "テスト"
3. 内容: 自由に書く
4. Cmd + P → "Git: Commit all changes"
5. GitHubで確認: https://github.com/nahato-Inc/IF-memo
```

**✅ GitHubにファイルが追加されていればOK！**

---

### Step 5: あなたの作業スペースを作成（7分）

#### 5-1. 自分のフォルダを作成

```
team/フォルダ内に自分の名前でフォルダを作成:

例:
- team/yamada/
- team/tanaka/
- team/suzuki/

※ guchiはサンプルなので参考にしてください
```

#### 5-2. フォルダ構造

```
team/あなたの名前/
├── README.md           ← あなたのダッシュボード
├── daily/              ← デイリーノート
├── projects/           ← プロジェクト管理
├── notes/              ← 技術メモ
└── code-snippets/      ← コードスニペット
```

#### 5-3. README.mdを作成

`team/guchi/README.md` をコピーして、名前を変更してください。

## 📖 基本的な使い方

### 毎日の作業フロー

#### 🌅 朝（業務開始時）

```
1. Obsidianを起動
2. team/あなたの名前/README.md を開く
3. Cmd + P → "Open today's daily note"
4. 今日のタスクを記入
```

#### 💼 作業中

```
- 会議 → meeting-note-template を使用
- コード → code-snippets/ に保存
- 学んだこと → notes/ に記録
- プロジェクト → projects/ で管理
```

#### 🌙 終業前

```
1. デイリーノートに作業記録を追記
2. Git同期を確認（自動で10分ごと）
3. 明日の予定を書く
```

```
✅ 既に設定済み！

```

## 🤝 チーム運用ルール

### ✅ やること

1. **毎日デイリーノートを書く**
   - 作業記録を残す
   - 学んだことを記録
   - 明日の予定を書く

2. **自分のフォルダで作業する**
   - `team/あなたの名前/` 配下
   - 他のメンバーのフォルダは参照のみ

3. **わかりやすいファイル名を付ける**
   - ❌ `無題のファイル.md`
   - ✅ `プロジェクトX進捗報告.md`

4. **タグを活用する**
   - `#自分の名前` タグを付ける
   - `#project` `#meeting` など

5. **リンクを活用する**
   - `[[関連ノート]]` でつなぐ
   - 情報を有機的に整理

### ❌ やらないこと

1. **他のメンバーのフォルダを編集しない**
   - 参照・リンクはOK
   - 編集は本人のみ

2. **機密情報を書かない**
   - パスワード
   - APIキー
   - 個人情報

3. **大きなファイルをコミットしない**
   - 画像: 10MB以下推奨
   - 動画: Vaultに入れない

4. **Gitを無効化しない**
   - バックアップは必須
   - チーム共有のため

## 🎨 便利な機能

### GitHub Alert記法

カラフルなブロックを使える：

```markdown
> [!NOTE]
> 青い枠の注記

> [!TIP]
> 緑の枠のヒント

> [!WARNING]
> 黄色い枠の警告

> [!IMPORTANT]
> 紫の枠の重要事項

> [!CAUTION]
> 赤い枠の注意
```

### コード実行

````markdown
```python
print("Hello, Obsidian!")
```
````
→ 「▶️」ボタンで実行可能！

### Dataviewで検索

```dataview
table file.mtime as 更新日時
from "team/あなたの名前"
sort file.mtime desc
limit 10
```
→ 自分の最近のノート10件が表示

## ❓ よくある質問

### Q1: Obsidianが起動しない
```
A: アプリケーションフォルダにあるか確認
   再インストールしてみる
```

### Q2: Gitが動かない
```
A: Cmd + P → "Git: Initialize a new repo"
   それでもダメなら、SSH鍵を再確認
```

### Q3: プラグインが表示されない
```
A: Obsidianを再起動
   設定 → コミュニティプラグイン → 有効化
```

### Q4: 誤って削除してしまった
```
A: Git履歴から復元可能
   先輩に相談してください
```

### Q5: 他のメンバーのノートを見たい
```
A: team/メンバー名/ を参照
   編集は本人のみ、閲覧は全員OK
```

### Q6: スマホで見たい
```
A: GitHub経由でWebから閲覧可能
   https://github.com/nahato-Inc/IF-memo
```

### Q7: 会社と自宅のPCで同期したい
```
A: 両方でリポジトリをクローン
   作業前: Cmd + P → "Git: Pull"
   作業後: 自動バックアップ
```
